# 抖音视频全自动工作流 - 使用指南

## 📋 功能概述

这个工作流可以帮你自动化制作抖音视频：

```
热点话题 → AI生成脚本 → 自动剪辑 → 抖音格式 → 批量输出
```

### 已完成功能

| 功能 | 状态 | 说明 |
|------|------|------|
| 热点库 | ✅ | 内置8大热门话题分类 |
| 自动剪辑 | ✅ | 按指定时长裁剪 |
| 抖音格式 | ✅ | 自动转9:16竖屏 |
| 批量生成 | ✅ | 一次生成多个视频 |
| 字幕生成 | ⚠️ | 需要修复（可选）|
| AI配音 | ⚠️ | 需要配置TTS（可选）|

## ❓ 常见问题

### Q: FFmpeg找不到？
A: 确保FFmpeg在以下路径之一：
- `C:\Users\ZhuanZ\.openclaw\workspace\ffmpeg\xxx\bin\ffmpeg.exe`
- 系统环境变量PATH中

### Q: 视频生成失败？
A: 检查：
1. 输入视频是否存在
2. 视频格式是否支持（MP4推荐）
3. 磁盘空间是否充足

### Q: 如何添加字幕？
A: 字幕功能需要修复，暂时可用剪映手动添加
