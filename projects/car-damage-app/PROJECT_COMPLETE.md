## 项目概述

基于AI视觉识别的车辆定损手机APP，采用"只换不修"原则，用户通过拍照上传车损图片，AI自动识别损伤部位、类型和程度，并生成配件报价单。

## 核心功能流程

```
用户拍照上传
    ↓
AI视觉识别 (YOLOv8)
    ↓
识别结果: 部位 + 损伤类型 + 严重程度
    ↓
只换不修规则判断
    ↓
配件匹配 (车型 + OE号 + 原厂价格)
    ↓
报价计算 (配件费 + 工时费)
    ↓
生成报价单 (PDF/分享)
```

## 只换不修规则

### 安全件（任何损伤必须更换）
- 前挡风玻璃
- 大灯（左/右）
- 尾灯
- 车门玻璃
- 结构件（纵梁等）

### 严重损伤（必须更换）
- 破裂、撕裂
- 穿孔
- 严重变形（面积>60%）

### 关联更换规则示例
```
前保险杠破裂 → 前保险杠总成 + 骨架 + 卡扣 + 内衬 + 雾灯 + 雷达探头
左前大灯破碎 → 左前大灯总成 + 支架 + 前保险杠(左侧) + 水箱上横梁
```

## 启动指南

### 1. 后端服务
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 数据库
```bash
# 安装PostgreSQL后
psql -U postgres -f database/schema.sql
psql -U postgres -f database/vehicles_full.sql
psql -U postgres -f database/parts_full.sql
psql -U postgres -f database/vehicle_parts.sql
psql -U postgres -f database/damage_rules.sql
```

### 3. AI模型训练
```bash
cd ai-models
python train_model.py
```

### 4. APP运行
```bash
cd frontend
flutter pub get
flutter run
```
