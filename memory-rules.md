# 会话记忆保存规则

## 每次会话结束时必须执行：

1. **检查是否有值得记录的内容**
   - 用户的待办事项、任务、计划
   - 重要的决定或偏好
   - 未完成的工作
   - 用户明确要求"记住"的事情

2. **写入 memory/YYYY-MM-DD.md**
   - 文件路径：`C:\Users\ZhuanZ\.openclaw\workspace\memory\YYYY-MM-DD.md`
   - 格式：日期标题 + 要点列表
   - 没有内容时也创建空文件标记会话

3. **同步更新 MEMORY.md**
   - 把长期需要记住的内容写入 MEMORY.md
   - 短期内容留在每日文件

## 示例格式：
```markdown
# 2026-03-30

## 待办事项
- [ ] 完成 xxx

## 重要信息
- 用户要求：保留对话内容

## 未完成
- xxx 待继续
```

## 检查点：
- [ ] 会话结束前检查是否需要保存
- [ ] 创建/更新当日 memory 文件
- [ ] 更新 MEMORY.md（如有长期内容）
